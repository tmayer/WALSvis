Description

This visualization shows the values for all WALS features by combining the geolocation of the respective languages with their genealogy in a sunburst visualization (Stasko and Zhang 2000). The map and the sunburst is enhanced with interactive functionalities. You can select a region in the world map and get only those languages spoken in that area displayed in the sunburst.
